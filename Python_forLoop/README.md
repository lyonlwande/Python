# Python For Loop Assignment

This folder contains 20 Python files that demonstrate basic list and `for` loop concepts. Each file corresponds to a single assignment task.

## Files and Tasks

1. `task1.py` - Display 5 student names using a `for` loop.
2. `task2.py` - Display 8 favorite foods one by one.
3. `task3.py` - Calculate the sum of a list of numbers using a `for` loop.
4. `task4.py` - Find the largest number in a list using a `for` loop.
5. `task5.py` - Find the smallest number in a list using a `for` loop.
6. `task6.py` - Count the number of students in a list using a loop.
7. `task7.py` - Print only even numbers from a list of 1 to 20.
8. `task8.py` - Print only odd numbers from a list of 1 to 20.
9. `task9.py` - Calculate the average mark for 10 student marks.
10. `task10.py` - Calculate total daily sales from a list of sales values.
11. `task11.py` - Count how many marks are 50 or above.
12. `task12.py` - Display products with numbered listing.
13. `task13.py` - Display marks that are greater than 80.
14. `task14.py` - Calculate total transport expenses for a week.
15. `task15.py` - Find the highest temperature from a weekly list.
16. `task16.py` - Count how many times `Mango` appears in a fruit list.
17. `task17.py` - Calculate the class total marks for 15 students.
18. `task18.py` - Display numbers greater than 100 from a list.
19. `task19.py` - Analyze student fees: total, highest, and lowest.
20. `task20.py` - Display supermarket stock report data and low-stock products.

## How to Run

Open a terminal in this folder and run any task file with Python:

```bash
python task1.py
python task2.py
# ...
python task20.py
```

## Notes

- Each task uses simple Python concepts: variables, lists, `for` loops, and `print()`.
- The code avoids functions, dictionaries, classes, and advanced topics.
- Files are designed to be easy to read and test individually.
