# Python Practice Exercises

This repository contains Python practice files focused on basic decision-making using `if`, `if-else`, `if-elif-else`, and nested `if` statements.

## Included exercises

- `Question1.py` — Pass or Fail check
- `Question2.py` — Bank loan qualification
- `Question3.py` — Uber driver bonus eligibility
- `Question4.py` — Data bundle discount decision
- `Question5.py` — Employee tax category assignment
- `Question6.py` — University admission decision
- `Question7.py` — ATM withdrawal nested if logic
- `Question8.py` — Scholarship eligibility nested if logic
- `Question9.py` — E-commerce delivery qualification nested if logic
- `Question10.py` — SACCO Loan Approval System
- `Task.py` — Additional custom practice task

## How to run

Run each exercise with Python from the repository root:

```powershell
python Question1.py
python Question2.py
python Question3.py
python Question4.py
python Question5.py
python Question6.py
python Question7.py
python Question8.py
python Question9.py
python Question10.py
```

## Notes

- Each question uses only variables and conditional statements.
- `Task.py` is an additional advanced nested-if practice exercise.
- Add output screenshots to a `screenshots/` folder for submission evidence.
